# Skyline Digital - Premium Web Solutions

Skyline Digital is a high-performance, aesthetically driven agency landing page designed to showcase modern web development capabilities and strategic digital solutions. This project features a primary landing page and a curated portfolio of specialized industry-specific sub-sites.

## 🚀 Key Features

- **Immersive User Experience:** Smooth scroll animations and a custom interactive cursor.
- **Modern Tech Stack:** Built with React, TypeScript, and Tailwind CSS for maximum performance and scalability.
- **Dynamic Content Sections:**
  - **Hero:** Impactful entry point with smooth typography.
  - **About:** Strategic overview of our digital growth and automation expertise.
  - **Services:** Detailed breakdown of web design, e-commerce, and AI automation solutions.
  - **Portfolio:** Interactive showcase of live projects.
  - **Pricing & Process:** Transparent roadmap and value-driven packages.
- **Responsive Design:** Fully optimized for all device sizes, from mobile to ultra-wide displays.

## 📁 Portfolio Sub-Pages

This repository includes a collection of specialized website templates:
- **Corporate Website:** Strategic solutions for global enterprises.
- **E-Commerce:** Modern headless commerce architecture.
- **Clinic:** Professional healthcare and wellness landing pages.
- **Gym Website:** High-energy sites for fitness and studios.
- **Portfolio:** Minimalist and effective personal branding.
- **Restaurant:** Refined culinary digital experiences.

## 🛠️ Tech Stack

- **Frontend:** React 18, TypeScript, Vite
- **Styling:** Tailwind CSS, PostCSS, Framer Motion
- **Icons:** Lucide React
- **Animations:** Custom CSS variables and scroll-triggered animations

## ⚙️ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Aryan31v/skyline-digital.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
